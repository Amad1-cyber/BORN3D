# Born 3D v0.0.1

**Created by: Ahmed Othmen**

Born 3D is a web-based 3D application featuring a "Pure" 3D editor for terrain and texture management, and a side game mode called "Born3D". Built with JavaScript and Three.js.

## Features
- 3D terrain (plane) editing
- Texture management via explorer
- Switch between editor and game mode
- Level save/load/overwrite with pin code (in progress)
- Prop and avatar placement

## Getting Started
Open index.html in your browser to launch the app.

## Project Structure
- index.html: Main entry point
- /src: JavaScript source code
- /assets: Textures and assets
- /styles: CSS styles
